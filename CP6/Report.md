1. **Тема**: Обработка последовательной файловой струкутуры на языке Си.    
2. **Цель работы**: Разработать последовательную структуру данных для представления простейшей базы данных на файлах СП Си в соответствии с заданным вариантом. Составить программу генерации внешнего нетекстового файла заданной структуры, содержащего представительный набор записей (15-20). Распечатать содержимое сгенерированного файла в виде таблицы и выполнить над ним заданное действие для параметров запроса ираспечатать результат.  
3. **Задание (Вариант №):** Вариант №11:  
     * **Содержимое файла**: Сведения о составе и комплектующих личных ПЭВМ в студенческой группе: фамилия, число и тип процессоров, объем памяти, тип видеоконтроллера(встроенный, внешний, AGP, PCI) и объем видеопамяти, тип (SCSI/IDE, ATA/SATA), число и ёмкость винчестеров, количество интегрированных контроллеров и внешних (периферийных) устройств, операционная система.
     * **Действия**: Перечислить все компьютеры студентов группы, платформа которых отлична от WINTEL.  
4. **Алгоритм решения:**  
     * **person.h**: Структура для представления данных о студенте: *surname* - фамилия студента, *count_proc* - число процессоров, *type_proc* - процессор, *memory* - объем памяти, *type_video* - видеоконтроллер, *videomemory* - объем видеопамяти, *type_vin* - винчестер, *count_vin* - количество винчестеров, *volume_vin* - ёмкость винчестера, *count_int* - количество интегрированных контроллеров, *count_per* - количество периферийных устройств, *system* - операционная система.
     * **generator.c**: Программа для считывания данных о студенте и его ПЭВМ и записи их в двоичный файл. Программа принимает один аргумент командной строки - имя файла, в который будут записаны данные. Если количество аргументов командной строки не равно 2, программа выводит сообщение о правильном использовании и завершает работу. Функция *readPerson* считывает данные о человеке в структуру *Person*. Она ожидает ввод строк, состоящих из 12 слов через пробел, и сохраняет их соответственно в поля структуры Person. Функция main открывает файл для записи данных в двоичном режиме (mode "wb"). После успешного открытия файла, она вступает в цикл, в котором происходит считывание данных о человеке с клавиатуры с последующей записью структуры *Person* в файл. Пустая строка символизирует конец ввода. После записи всех данных программа закрывает файл и завершает свою работу.
     * **program.c**: Программа для вывода информации из двоичного файла в виде таблицы. Программа принимает 2 аргумента: название двоичного файла и параметр. При некорректном вводе аргументов программа выдает соответствующее сообщение. Если введен параметр *-f*, то программы выводит данные о всех студентах и их ПЭВМ. Если введен параметр *-p*, то программа выводит информацию о студентах, платформа которых отлична от WINTEL.

    Person p;
    FILE *file = NULL;

    if (count_arg != 2)
    {
        printf("Usage: %s filename\n", args[0]);

        return 1;
    }

    file = fopen(args[1], "wb");

    if (file == NULL)
    {
        printf("Openfile error\n");

        return 1;
    }

    int result;
	printf("Enter 12 words for a person (or leave blank line to exit):\n");
    while ((result = readPerson(&p)) == 1) {
        fwrite(&p, sizeof(p), 1, file);
    }

    fclose(file);

    return 0;
}

int readPerson(Person *p)
{
    char input[100]; // Максимальная длина строки - 200 символов
    char *word;
        
    if (fgets(input, sizeof(input), stdin) != NULL) {
        if (strlen(input) <= 1) {
            return 0; // Пустая строка
        }

        word = strtok(input, " \n");
        strcpy(p->surname, word);

        for (int i = 0; i < 11; i++) {
            word = strtok(NULL, " \n");
            if (word == NULL) {
                printf("Not enough words entered!\n");
                return -1; // Недостаточно слов
            }
            switch(i) {
                case 0:
                    strcpy(p->count_proc, word);
                    break;
                case 1:
                    strcpy(p->type_proc, word);
                    break;
                case 2:
                    strcpy(p->memory, word);
                    break;
                case 3:
                    strcpy(p->type_video, word);
                    break;
                case 4:
                    strcpy(p->videomemory, word);
                    break;
                case 5:
                    strcpy(p->type_vin, word);
                    break;
                case 6:
                    strcpy(p->count_vin, word);
                    break;
                case 7:
                    strcpy(p->volume_vin, word);
                    break;
				case 8:
                    strcpy(p->count_int, word);
                    break;
				case 9:
                    strcpy(p->count_per, word);
                    break;
				case 10:
                    strcpy(p->system, word);
                    break;
            }
        }
        return 1; 
    } 
    return -1; 
}

        printf("Usage: %s filename -f/-p\n", args[0]);
        return 1;
    }

    FILE *file = fopen(args[1], "rb");
